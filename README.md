# MyCyanGlasses

Android companion app for HeyCyan smart glasses, built with the official glasses SDK.

## Features

- **Bluetooth Connection**: Scan and connect to HeyCyan glasses via BLE
- **Camera Control**: Take photos, record videos, record audio
- **Device Management**: Sync time, check battery, view firmware version
- **Media Gallery**: View downloaded photos, videos, and audio files

## Related

This is the Android implementation for the [NTHU AI Glasses App Project](https://github.com/Buddy-Lu/NTHU-AI-glasses-app-project-2026), which contains UI mockups in React Native.

## Requirements

- Android Studio
- Android device with Bluetooth LE support
- HeyCyan smart glasses

## SDK

Uses `glasses_sdk_20250723_v01.aar` for glasses communication.
